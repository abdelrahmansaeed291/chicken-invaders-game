# chicken invaders

### The environment :
   + The player is positioned in the middle of the screen at the start of the game,
   + Health bar displayed on screen for the enemy,
   + 3 lives displayed on screen for the player.
### player: The player starts at a fixed position in the middle of the screen and is controlled by the keyboard keys to move in 4 directions.
### enemy: Starts at the top of the screen and then moves a continuous motion from left to right , the enemy moves to the right and when it
reach the border of the screen it changes direction to move to the left .
### power-ups:
   + if the player was to collide with one or more of the bullets from the enemy no lives will be lost,
   + Power-ups always appear randomly at various positions and remain fixed in position until acquired by the player, in this case the power up disappears .
   + The effect of the power up lasts for a few seconds then disappears .

### How the game finishes 
 + if the enemy was killed .
 + if three bullets hit the player .
 
 
### How to play.
  + the player can move with the four basic arrows .
  + the player can shut with the space button .
  
### How to run the game 
  + pull the project and run the file OpenGLMeshLoader.sln .
  
### Technologies
  + C and C++ (openGL).
